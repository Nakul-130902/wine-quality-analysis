# 🍷 Wine Quality EDA Project

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on the Red Wine Quality dataset to understand how different physicochemical properties influence the quality of wine.

---

## 🎯 Objective

* Analyze the dataset using visualization techniques
* Identify key factors affecting wine quality
* Understand relationships between different features

---

## 📊 Dataset Information

* Dataset: Red Wine Quality Dataset
* Number of Rows: 1599
* Number of Columns: 12
* Target Variable: `quality`

---

## 🛠️ Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📈 Exploratory Data Analysis (EDA)

### ✔️ Steps Performed:

* Data loading and inspection
* Handling missing values
* Statistical summary
* Distribution analysis (Histograms)
* Correlation analysis (Heatmap)
* Relationship analysis (Scatter plots)
* Group comparison (Boxplots)

---

## 🔍 Key Insights

* Alcohol has a strong positive relationship with wine quality
* Volatile acidity negatively impacts quality
* Fixed acidity shows weak correlation with quality
* Most wines fall under quality range 5–6 (imbalanced dataset)

---

## 📌 Conclusion

EDA reveals that alcohol is one of the most important features influencing wine quality, while some features like fixed acidity have minimal impact. Visualization helped in identifying patterns, trends, and outliers in the dataset.
